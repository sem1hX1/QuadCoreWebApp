# QuadCore 🚀

QuadCore, satıcılar için tasarlanmış AI destekli bir ticaret asistanıdır.

## Özellikler
- **Alırken Kazan:** Toptancı fiyatlarını karşılaştırarak en uygun tedarikçiyi bulur.
- **Satarken Kazan:** Perakende pazarındaki rakipleri analiz ederek en ideal satış fiyatını önerir.
- **AI Analizi:** Ürün fotoğraflarından veya isimlerinden pazar taraması yapar.

## Teknoloji Yığını
- **Backend:** Python, FastAPI
- **Frontend:** React, Tailwind CSS
- **Veri İşleme:** BeautifulSoup, Pandas

## Kurulum
1. `cd backend`
2. `pip install -r requirements.txt`
3. `uvicorn app.main:app --reload`
